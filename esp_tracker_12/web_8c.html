void start_webserver ( void )

Starts an HTTP web server and registers a handler for the "/logs" URI.

This function initializes and starts an embedded HTTP server on the ESP32. It configures the server with default settings and then registers a specific URI handler (log_page_get_handler) to serve a dynamic log page when clients access the "/logs" endpoint.

void wifi_init_sta ( void )

Initializes the ESP32 as a Wi-Fi station and connects to an access point.

This function sets up the Wi-Fi in station mode, registers event handlers for connection management, attempts to connect to a specified Wi-Fi network, and waits for the connection outcome. It logs success or failure and then unregisters the event handlers.
