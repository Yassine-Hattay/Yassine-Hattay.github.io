This function processes touch coordinates to determine which keyboard key was pressed, then performs the appropriate action based on the key and current application context.

◆ init_XPT2046()

void init_XPT2046 ( )

Initializes the XPT2046 touch controller.

This function configures the necessary GPIO pins for communication with an XPT2046 touch controller and sends an initial control byte to prepare it for operation. It sets up the MISO and PIRQ pins as inputs, the SS pin as an output, and enables an interrupt on the PIRQ pin for touch detection.

◆ recieve_touch_data()

uint16_t * recieve_touch_data ( int r)

Receives a specified number of data points and extracts a 12-bit value.

This function reads 'r' data points using an assumed recive() function, stores them in a received array, and then attempts to construct a 12-bit unsigned integer from the first two received bytes.

Parameters
rThe number of data points to receive. Must be a positive integer.
Returns
A pointer to a static uint16_t variable containing the extracted 12-bit value. Returns NULL if r is invalid, or 0 if there's not enough data to form a 12-bit value.
